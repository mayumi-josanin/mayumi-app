使い始める前に必ず確認したいこと

このアプリでいちばん多いトラブルは、ブラウザで先に登録したあと、ホーム画面に追加してもう一度新規登録してしまうことです。ここを先に確認しておくと、同じお名前で別の会員IDができる事故を防ぎやすくなります。

重複登録を防ぐ基本ルール
ホーム画面のアイコンから使いたい方は、先にホーム画面へ追加してから、そのアイコンから会員登録してください。
すでに会員登録済みの方は、新しい入口から開いたときに 新規登録し直さず、「データの引き継ぎ・復元はこちら ↺」からお入りください。

iPhone の方

  1. Safari でアプリのページを開きます。
  2. 共有ボタンを押します。
  3. 「ホーム画面に追加」を選びます。
  4. 表示された場合は「Webアプリとして開く」をオンにして追加します。
  5. 追加後はホーム画面のアイコンから開いて使ってください。

Android の方

  1. Chrome でアプリのページを開きます。
  2. 右上の「︙」メニューを押します。
  3. 「ホーム画面に追加」または「アプリをインストール」を選びます。
  4. 画面の案内に従って追加します。
  5. 追加後はホーム画面のアイコンから開いて使ってください。
すでに登録済みの方
ホーム画面に追加したあと、情報が引き継がれていないように見えても、まずは新規登録ではなく復元を試してください。
同じ開き方で使う
「ホーム画面のアイコンから使う」「Safariで開く」など、使う入口をできるだけ統一するとトラブルが減ります。

初回登録の流れ

はじめて使う方は、プロフィール登録とパスコード設定を行います。登録済みの方でも、パスコード未設定の場合は最初に設定画面が出ます。

新しく登録する方

  1. アプリを開き、「はじめる」を押します。
  2. プロフィール登録画面で お名前 を入力します。お名前は必須です。
  3. 必要に応じて、電話番号・生年月日・住所・アイコン画像・背景画像を入力します。
  4. 4桁または6桁の数字でパスコードを設定します。
  5. 「登録してアプリを始める」を押すと、ホーム画面へ進みます。

すでに登録済みの方で、まだパスコードを設定していない場合

  1. アプリを開くと、パスコード設定の案内が表示されます。
  2. 4桁または6桁の数字でパスコードを設定します。
  3. 設定後は通常どおりホーム画面へ進めます。
必須
お名前、パスコード
あとで変更できる項目
電話番号、生年月日、住所、アイコン画像、背景画像
登録できないとき
お名前が空欄だと登録できません。
パスコードは 4桁または6桁の数字のみ です。英字や5桁は使えません。

起動時パスコードとログインの使い方

パスコードは、アプリを安全に開くための数字です。いちど設定したあと、毎回入力するかどうかはお客様ご自身で選べます。

アプリを開くとき

  1. 起動時パスコードがオンの場合は、ログイン画面が表示されます。
  2. 設定した4桁または6桁の数字を入力します。
  3. 正しく入力するとホーム画面に入れます。
  4. ログイン画面には「データの引き継ぎ・復元はこちら ↺」も表示されます。

毎回パスコードを入れるかどうかの切り替え

  1. マイページを開きます。
  2. 「ログイン時のパスコード」を確認します。
  3. ボタンを押して、オン / オフを切り替えます。
  4. オンにすると次回から毎回入力、オフにすると次回から省略できます。

パスコードを変更したいとき

  1. マイページの「🔐 パスコードを変更」を押します。
  2. 現在のパスコードを確認します。
  3. 新しい4桁または6桁の数字を設定します。

パスコードを忘れたとき

  • ログイン画面の「パスコードを忘れた場合の再設定はこちら」から進めます。
  • お名前・電話番号・生年月日を入力し、新しいパスコードを設定します。
  • 再設定後は、その新しいパスコードで入れます。
注意
パスコードをオフにしても、プロフィールや注文履歴は消えません。
ただし端末を他の方も使う場合は、起動時パスコードをオンにしておくと安心です。

データ引き継ぎ・復元・機種変更の方法

機種変更や再インストールのときは、新規登録ではなく復元 を使います。復元には「引き継ぎコードを使う方法」と「登録情報で復元する方法」があります。

引き継ぎコードを発行する方法

  1. 旧端末でマイページを開きます。
  2. 「↺ 引き継ぎコードの発行」を押します。
  3. 表示された8桁のコードを控えます。
  4. 有効期限を確認します。コードは 1回限り・1週間有効 です。

新しい端末で復元する方法

  1. アプリを開き、「データの引き継ぎ・復元はこちら ↺」へ進みます。
  2. 引き継ぎコードがある場合は、引き継ぎコード + 新しい端末で使うパスコード を入力します。
  3. 引き継ぎコードがない場合は、お名前 を入力し、さらに 電話番号・生年月日・現在のパスコードのうち1つ以上 を入力します。
  4. 「ログインして復元する ↺」を押します。
  5. 復元が成功すると、そのままホーム画面へ進みます。

再インストール前後のおすすめ確認

  • 旧端末が使えるうちに引き継ぎコードを発行しておく
  • 新端末では新規登録しない
  • 復元後はマイページで会員IDとお名前を確認する
  • 必要に応じて通知設定をオンにし直す
  • 注文履歴や特典状況が見えるか確認する
復元に失敗しやすい例
新しい端末で先に新規登録してしまう / 引き継ぎコードの有効期限が切れている / お名前が登録時と違う表記になっている / 現在のパスコードではなく新しいパスコードを入力している、などで復元できないことがあります。

ホーム画面の使い方

ホーム画面はアプリの入口です。スタンプ、最新情報、メニュー、外部リンクをひとまとめに確認できます。

スタンプを取得する 院内のQRコードを読み取ってスタンプをためます。
現在のスタンプカード 今のスタンプ数、何枚目のカードか、あと何個で達成かを確認できます。
おすすめ商品 人気商品をすぐに確認できます。タップすると商品詳細が開きます。
最新のお知らせ・ブログ 直近のNEWSが表示されます。「もっと見る」でNEWSページへ進めます。
まゆみ助産院のメニュー 施術やサービス内容の一覧入口です。予約確定は公式LINEから行います。
公式サイト・SNS 公式ホームページ、Instagram、Facebook、公式LINE、まゆみのブログへ進めます。

ホーム画面を開くことが多い場面

  • 来院時にスタンプを取りたいとき
  • 新しいお知らせがないか確認したいとき
  • メニュー一覧へ進みたいとき
  • 公式LINEやSNSへ移動したいとき

スタンプ取得とカメラの使い方

来院時はホーム画面からスタンプを取得します。読み取りにはカメラ許可が必要です。

スタンプ取得の流れ

  1. ホーム画面の「📷 カメラを起動して読み取る」を押します。
  2. アプリ内で「カメラを使用します。許可してください」という案内が出たら内容を確認します。
  3. 続けて表示される iPhone / Android の確認画面で「許可」を押します。
  4. 院内のQRコードをカメラに向けます。
  5. 読み取りに成功するとスタンプが1つ追加されます。
スタンプ取得のルール
スタンプは 1日1回まで です。
同じ日に再度読み取ると「本日はすでにスタンプを取得済みです」と表示されることがあります。

カメラが起動しないとき

  • 確認画面で「許可しない」を選んでいないか確認してください。
  • iPhone と Android で設定場所が異なるため、次のページの詳しい手順を確認してください。
  • 設定が終わったら、いちど閉じてから再度「📷 カメラを起動して読み取る」を押してください。
  • 暗い場所では読み取りにくいため、明るい場所で試してください。

07 補足

📷 カメラが起動しないときの詳しい設定方法

セキュリティ設定により、カメラへのアクセスがブロックされていることがあります。お使いのスマートフォンに合わせて、以下の手順でカメラの「許可」をお願いいたします。

【iPhoneをご利用のお客様】

ホーム画面に追加したアイコンから起動している場合、大元となる Safari の設定を変更する必要があります。アプリ個別の設定画面に「通知」しか表示されないのは正常です。

設定手順
  1. iPhone の 「設定」 を開きます。
  2. 少し下にスクロールし、「Safari」 を押します。
  3. 「Webサイトの設定」 の中にある 「カメラ」 を押します。
  4. 「許可」 にチェックを入れます。

【Androidをご利用のお客様】

Android は、機種や開き方によって設定先が少し違います。ご自身の画面の状態に近い方を確認してください。

① 設定画面に「通知」しか見当たらない場合
  1. スマートフォンの 「設定」 を開きます。
  2. 「アプリ」 を押し、今回開いているアプリまたはアイコンを選びます。
  3. 「権限」 を押します。
  4. 「カメラ」 を選び、「アプリの使用中のみ許可」 に変更します。
② アイコンの右下に Chrome のマークがついている場合
  1. 通常の 「Google Chrome」 アプリを開きます。
  2. 右上の 「︙」 を押し、「設定」 を選びます。
  3. 「サイトの設定」 → 「カメラ」 の順に押します。
  4. 一覧からスタンプラリーのサイトを探して押し、「許可」 に変更します。
設定が終わったら
設定が完了したら、一度画面を閉じて、もう一度ホーム画面のアイコンからアプリを開き直し、カメラが起動するかお試しください。

特典ガチャと特典取得状況

スタンプが10個たまると特典ガチャを回せます。結果はマイページに保存されます。

10個たまったあと

  1. ホーム画面に「🎯 特典ガチャを回す」が表示されます。
  2. ボタンを押してガチャを回します。
  3. 結果が保存されると、ホーム画面に「🌸 新しいスタンプカードを取得」が表示されます。
  4. そのボタンを押すと次のスタンプカードへ進めます。

特典取得状況の見方

  1. マイページの「🎁 特典取得状況」を開きます。
  2. 獲得済みの特典、使用済みかどうか、有効期限を確認します。
  3. 使える特典がある場合は「使用する」を押します。
  4. 確認画面で確定すると、受付で見せる完了画面が表示されます。

特典利用時の注意

  • 特典は達成当日から使用できます。
  • 有効期限は獲得日から1か月です。
  • いちど「使用する」を押した特典は元に戻せません。
  • 特典の受け取りは受付へ直接お問い合わせください。
よくある見落とし
スタンプが10個たまっても、ガチャを回す前には新しいカードを開始できません。
まず特典ガチャを回してから、次のカードへ進んでください。

ショップ・カート・注文方法

ショップでは商品説明を見て、カートに入れて注文できます。注文は院内受け取り・現金払いです。

商品を見る流れ

  1. 下部メニューの「ショップ」を開きます。
  2. 商品をタップして詳細画面を開きます。
  3. 商品説明や説明画像がある場合は内容を確認します。
  4. 個数を「＋」「－」で調整します。

カートに入れて注文する流れ

  1. 「カートに追加する」を押します。
  2. 画面上部の「🛒」を押してカートを開きます。
  3. 商品名、個数、合計金額を確認します。
  4. 必要に応じて個数変更や削除を行います。
  5. 「ご注文を確定する」を押すと注文完了です。
受け取り方法 院内受け取り
支払い方法 現金払い
注文できる条件 プロフィール登録が必要です。未登録の場合は先に登録画面が表示されます。

ショップで覚えておきたいこと

  • 商品によって説明画像や詳しい説明が表示されます。
  • 商品内容は更新で変わることがあります。気になるときは上部の「🔄」で更新してください。
  • 注文完了後は、マイページの注文履歴から状況を確認できます。

注文履歴の見方

注文後の確認は、マイページの「📋 ご注文履歴」で行います。

  1. マイページを開きます。
  2. 「📋 ご注文履歴」を下へスクロールして確認します。
  3. 注文番号、注文日、商品名、個数、合計金額、状態を確認します。
  4. 受付中の注文には「キャンセルする」が表示されることがあります。
  5. 受け取り前の注文には「受け取りました」が表示されることがあります。

状態の見方

  • 受付中 まだ受け取り前の状態です。
  • 受取済 受け取り済みとして記録されています。
  • キャンセル済 キャンセル済みの注文です。
履歴が見つからないとき
まず上部の「🔄」で更新し、それでも出ない場合はマイページを開き直してください。
「再読み込み」ボタンが表示されている場合は、そのボタンも試してください。

こんなときに使います

カレンダーの見方

診療日やイベント予定を確認したいときは、カレンダーページを使います。

  1. 下部メニューの「カレンダー」を開きます。
  2. 画面上部の「◀」「▶」で月を切り替えます。
  3. 日付をタップすると、その日のイベントが表示されます。
  4. 「今月のイベント一覧」から、その月の予定をまとめて確認できます。

トラブル対応集 1
登録・ログイン・復元まわり

新規登録できない 1. お名前が入力されているか確認してください。
2. パスコードが4桁または6桁の数字だけになっているか確認してください。
3. 通信状態を確認し、上部の「🔄」や再起動後に再度お試しください。
同じ名前で別の会員IDになった 1. ブラウザ版とホーム画面版で別登録した可能性があります。
2. これ以上新規登録し直さず、マイページで会員IDを確認してください。
3. 受付または公式LINEへ、お名前と確認できた会員IDをお知らせください。
パスコードが通らない 1. 4桁または6桁の数字だけを入れているか確認してください。
2. 古いパスコードと新しいパスコードを間違えていないか確認してください。
3. どうしても入れない場合は「パスコードを忘れた場合の再設定はこちら」から再設定してください。
起動時パスコードをやめたい 1. マイページを開きます。
2. 「ログイン時のパスコード」を押してオフに切り替えます。
3. 次回から起動時入力を省略できます。
パスコードを忘れた 1. ログイン画面またはログイン・引き継ぎ画面の再設定導線から進みます。
2. お名前・電話番号・生年月日を入力します。
3. 新しい4桁または6桁の数字を設定してください。
復元できない 1. 引き継ぎコードがある場合は、8桁すべて正しく入力してください。
2. 引き継ぎコードがない場合は、お名前に加えて電話番号・生年月日・現在のパスコードのうち1つ以上が必要です。
3. 新しい端末用のパスコードも必ず入力してください。
4. すでに新規登録してしまった場合は、それ以上登録し直さずご相談ください。
引き継ぎコードが使えない 1. 有効期限が切れていないか確認してください。
2. コードは1回限りです。すでに使ったコードは再利用できません。
3. 旧端末が使えるなら、マイページで新しい引き継ぎコードを発行してください。

トラブル対応集 2
カメラ・スタンプ・注文・通知・表示まわり

カメラが起動しない 1. カメラ許可の確認画面で「許可」を選んだか確認してください。
2. iPhone は「設定 → Safari → カメラ → 許可」を確認してください。
3. Android は「設定 → アプリ → 権限 → カメラ」、または Chrome の「設定 → サイトの設定 → カメラ」を確認してください。
4. 設定が終わったら、ホーム画面のアイコンから開き直して再度お試しください。
5. 詳しい手順は前の「📷 カメラが起動しないときの詳しい設定方法」をご覧ください。
QRを読み取れない 1. QRコード全体が画面に入るようにしてください。
2. 暗い場所では読み取りにくいため、明るい場所でお試しください。
3. カメラレンズが汚れていないか確認してください。
スタンプが増えない 1. その日のスタンプをすでに取得していないか確認してください。
2. 10個達成後で、先に特典ガチャを回す必要がないか確認してください。
3. プロフィール登録済みか確認してください。
4. 上部の「🔄」で更新してから再確認してください。
特典ガチャが出ない 1. スタンプが10個たまっているか確認してください。
2. すでにそのカードでガチャを回している場合は、「🌸 新しいスタンプカードを取得」が出ていないか確認してください。
3. 上部の「🔄」で更新してください。
注文できない 1. プロフィール登録が完了しているか確認してください。
2. カートに商品が入っているか確認してください。
3. 通信状況を確認し、再度お試しください。
注文履歴が出ない 1. マイページを開き直してください。
2. 「再読み込み」ボタンがあれば押してください。
3. 上部の「🔄」で更新してください。
4. それでも出ない場合は通信状況を確認してください。
通知が届かない 1. マイページの通知設定がオンか確認してください。
2. iPhone / Android 本体側の通知許可を確認してください。
3. ブラウザ版の場合は、ブラウザ側の通知許可も確認してください。
4. 一度オフにして、もう一度オンにすると直る場合があります。
NEWSやお知らせ一覧が古い 1. 上部の「🔄」を押して最新情報を取り直してください。
2. NEWSは記事ページ、お知らせ一覧は更新まとめページです。見ている場所が合っているか確認してください。
3. それでも反映されない場合は、アプリを閉じて開き直してください。
アプリが重い / 固まる / 画面が変わらない 1. いったんアプリを閉じて開き直してください。
2. 端末を再起動してください。
3. 通信が不安定な場所では読み込みに時間がかかることがあります。
4. 「アップデートが必要です」と出る場合は最新バージョンへ更新してください。
